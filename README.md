## Usage

### Prerequisites

- node
- npm

### Getting started

Clone the repository to your local machine

```
$ git clone https://github.com/BalintMendli/weather-app.git
```

Change into the project directory

```
$ cd weather-app
```

Install dependencies

```
$ npm install
```

Start development server

```
$ npm run dev
```

### Run the test suite

```
$ npm run cy:open
```
